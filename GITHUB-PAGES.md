## ⚙️ Настройка GitHub Pages (для каждого проекта)

### Шаг 1: Откройте репозиторий
Перейдите в репозиторий проекта на GitHub (ссылки выше)

### Шаг 2: Откройте Settings
Нажмите на вкладку **Settings** в верхней части репозитория

### Шаг 3: Перейдите в Pages
В левом меню найдите и нажмите **Pages**

### Шаг 4: Настройте Source
- **Source**: Deploy from a branch
- **Branch**: main → `/ (root)`
- Нажмите **Save**

### Шаг 5: Повторите для всех 6 проектов!

После включения GitHub Pages:

## 🔧 Если что-то не работает

1. **Проверьте Actions**: В каждом репозитории вкладка Actions должна быть зелёной
2. **Обновите страницу**: GitHub Pages может обновляться до 5 минут
3. **Проверьте настройки**: Убедитесь, что выбрана ветка `main` и папка `/ (root)`
