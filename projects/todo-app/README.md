# To-Do List App

Приложение для управления задачами с возможностью добавления, редактирования, фильтрации и сохранения в localStorage.

## 🚀 Технологии

- React 18
- TypeScript
- CSS Modules
- LocalStorage API

## ✨ Функции

- ✅ Добавление новых задач
- ✅ Отметка выполненных задач
- ✅ Удаление задач
- ✅ Фильтрация (все/активные/завершённые)
- ✅ Сохранение в localStorage
- ✅ Подсчёт оставшихся задач
- ✅ Очистка завершённых задач

## 🛠️ Установка

```bash
npm install
```

## 🏃 Запуск

```bash
npm run dev
```

## 📦 Сборка

```bash
npm run build
```

## 📱 Скриншот

Приложение имеет современный градиентный дизайн с фиолетово-синей цветовой схемой.
