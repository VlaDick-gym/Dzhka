# Expense Tracker

Трекер личных расходов и доходов с визуализацией данных по категориям.

## 🚀 Технологии

- React 18
- TypeScript
- CSS3
- LocalStorage API
- Recharts (опционально для графиков)

## ✨ Функции

- ✅ Добавление доходов и расходов
- 📊 Категории операций
- 💰 Подсчёт баланса
- 📈 Визуализация расходов по категориям
- 💾 Сохранение в localStorage
- 🗑️ Удаление операций
- 📱 Адаптивный дизайн

## 🛠️ Установка

```bash
npm install
```

## 🏃 Запуск

```bash
npm run dev
```

## 📦 Сборка

```bash
npm run build
```
