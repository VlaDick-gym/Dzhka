# Recipe Finder

Приложение для поиска рецептов блюд с интеграцией TheMealDB API.

## 🚀 Технологии

- React 18
- TypeScript
- Tailwind CSS
- TheMealDB API (демо-режим)

## ✨ Функции

- 🔍 Поиск рецептов по названию
- 📱 Просмотр деталей рецепта
- 🏷️ Категории и теги
- 🌍 Кухни мира
- 📺 Ссылки на видео-рецепты
- 🎨 Адаптивный дизайн

## 🛠️ Установка

```bash
npm install
```

## 🏃 Запуск

```bash
npm run dev
```

## 📦 Сборка

```bash
npm run build
```

## 📝 Примечание

Приложение работает с демо-данными. Для полного функционала зарегистрируйтесь на [TheMealDB](https://www.themealdb.com/api.php) и добавьте API ключ.
