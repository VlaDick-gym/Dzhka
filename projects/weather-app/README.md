# Weather App

Приложение с прогнозом погоды для любого города мира с интеграцией OpenWeatherMap API.

## 🚀 Технологии

- React 18
- TypeScript
- CSS3
- Fetch API
- OpenWeatherMap API (демо-режим)

## ✨ Функции

- 🔍 Поиск погоды по названию города
- 🌡️ Отображение текущей температуры
- 💨 Информация о ветре и влажности
- 😊 Ощущаемая температура
- 🎯 Быстрый выбор популярных городов
- 📱 Адаптивный дизайн

## 🛠️ Установка

```bash
npm install
```

## 🏃 Запуск

```bash
npm run dev
```

## 📦 Сборка

```bash
npm run build
```

## 📝 Примечание

Приложение работает в демо-режиме с предзагруженными данными. Для получения реальных данных зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/api) и добавьте свой API ключ.
