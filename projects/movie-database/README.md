# Movie Database

Каталог фильмов с поиском, фильтрами и рейтингами.

## 🚀 Технологии

- React 18
- TypeScript
- CSS3
- TMDB API (демо-режим)

## ✨ Функции

- 🔍 Поиск по названию и режиссёру
- 🎭 Фильтр по жанрам
- ⭐ Фильтр по рейтингу
- 📱 Просмотр деталей фильма
- 🎨 Адаптивный дизайн
- 📊 Счётчик результатов

## 🛠️ Установка

```bash
npm install
```

## 🏃 Запуск

```bash
npm run dev
```

## 📦 Сборка

```bash
npm run build
```

## 📝 Примечание

Приложение работает с демо-данными. Для полного функционала зарегистрируйтесь на [TMDB API](https://www.themoviedb.org/documentation/api) и добавьте API ключ.
