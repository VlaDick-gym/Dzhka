## ❗ Почему проекты не работают

GitHub Actions работают (зелёные галочки ✓), но **GitHub Pages не включены в настройках репозиториев**.

**Без ручного включения Pages — проекты НЕ БУДУТ работать!**

## ⚡ БЫСТРАЯ ИНСТРУКЦИЯ (3 минуты на каждый проект)

### Для КАЖДОГО из 6 проектов выполните:

#### Шаг 1: Откройте настройки Pages
Кликните по ссылке (откроется в новой вкладке):

1. https://github.com/VlaDick-gym/todo-app/settings/pages
2. https://github.com/VlaDick-gym/weather-app/settings/pages
3. https://github.com/VlaDick-gym/recipe-finder/settings/pages
4. https://github.com/VlaDick-gym/github-profile-viewer/settings/pages
5. https://github.com/VlaDick-gym/expense-tracker/settings/pages
6. https://github.com/VlaDick-gym/movie-database/settings/pages

#### Шаг 2: Включите Pages

На странице настроек:

1. Найдите раздел **"Build and deployment"**
2. **Source**: Выберите **GitHub Actions** (НЕ "Deploy from a branch"!)
3. Сохраните (если есть кнопка Save)

#### Шаг 3: Проверьте

Через 1-2 минуты после включения:
- Вверху страницы увидите: **"Your site is live at..."**
- Сайт станет доступен по ссылке
