# 🔴 СРОЧНО: Как включить GitHub Pages для проектов

## ❗ Почему проекты не работают

Код загружен в репозитории ✅, но **GitHub Pages не включены**. Без этого проекты не будут доступны в интернете.

---

## 📋 Пошаговая инструкция (ПОВТОРИТЬ ДЛЯ КАЖДОГО ИЗ 6 ПРОЕКТОВ)

### Шаг 1: Откройте репозиторий

Перейдите по ссылке:
- https://github.com/VlaDick-gym/todo-app

### Шаг 2: Откройте Settings

В верхней части репозитория нажмите на вкладку **Settings** (шестерёнка)

### Шаг 3: Найдите Pages

В левом боковом меню прокрутите вниз до раздела **Code and automation**
Нажмите на **Pages**

### Шаг 4: Включите GitHub Pages

На странице Pages:

1. В разделе **Source** выберите: **Deploy from a branch**

2. В разделе **Branch**:
   - Выберите ветку: **master** (не main!)
   - Папка: **/(root)**
   
3. Нажмите кнопку **Save**

### Шаг 5: Проверьте

После сохранения вы увидите сообщение:
> "Your site is live at https://vladick-gym.github.io/todo-app/"

Это займёт 1-3 минуты.

## 🚀 Быстрый способ (перейдите по ссылкам выше и настройте)

1. Кликните на ссылку репозитория (например, todo-app/settings/pages)
2. Выберите **master** branch
3. Нажмите **Save**
4. Повторите для остальных 5 проектов

2. Прокрутите до раздела **Проекты**
3. Нажмите кнопку **Запустить** на любом проекте
4. Проект должен открыться в новой вкладке

## 🔧 Если не работает

### Проверьте Actions:
1. Откройте репозиторий проекта
2. Перейдите во вкладку **Actions**
3. Должна быть зелёная галочка ✓
4. Если жёлтая или красная — подождите или проверьте настройки

### Проверьте Pages:
1. Settings → Pages
2. Убедитесь, что выбрано **master** branch
3. Сайт должен быть "live"

### Подождите:
GitHub Pages обновляются 2-5 минут после включения.
